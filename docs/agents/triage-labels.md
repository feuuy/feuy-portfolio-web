# Triage labels

This repository uses the default label vocabulary for issue triage.

## Canonical mapping

- `needs-triage`: maintainer needs to evaluate
- `needs-info`: waiting on reporter
- `ready-for-agent`: fully specified and ready for an AFK agent
- `ready-for-human`: requires human implementation
- `wontfix`: will not be actioned

## Notes for agent skills

Use these exact label strings when triaging issues.

If a label does not exist yet in GitHub, create it or ask the maintainer to create it, depending on the workflow being used.
